How i developed my Angular app

Like always, before i get into how i developed my app specifically angular, i think its always important to give a background of what Angular really is. Definition, Angular is a platform and framework for building single-page client applications using HTML and TypeScript.The architecture of an angular application relies on certain fundamental concepts. The basic building blocks of an angular framework are angular components that are organzied into things called Ngmodules. Those collect related code into functional sets. That all leads into what an angualr app is. Now that we went over the basics of Angular, lets get into how i developed my app.

Step 1:

The first Step I did was visit W3Schools Angular Tutorialand clicked on the three lines in the top left hand corner. From there, you have a variety of different of options to choose from that give a tutorial on different angular functions. I went through a bunch until i found the function i wanted to use. The tutorial that i chose wasAngularJS Applications.

From that tutorial i scrolled down until i saw "AngularJS Applications". Take a look at the code right here

That code uses modules to define AngularJS applications. The Controllers control the applications and the ng-app directive defines the application, the ng-controller directive defines the controller.

Step 3:

What i had to do to modify this code to my liking was i wanted to add the option to add your middle name into the input type so your middle name pops out too. so i went into the code and added Middle Name input type="text" ng-model= "middleName">br> between firstName input and lastName input.

Step 4:

My next step was i had to then manipulate Full Name: {{firstName + " " + lastName}} so that when i typed into the input, it would show up on the page. i ended up changing it to Full Name: {{firstName + " " + middleName + " " + lastName}}

Step 5:

I then had to change the app.controller. I added $scope.middleName = ""; between app.controller first and last name.

Step 6:

Just to jazz things up a bit more i decided to add a button that you can press once your done typing your name into the name generator that pops up a notification. the code for it was. button type="button" onclick="alert('Congratulations you typed your name!')">Click when done button> It was super simple but added in some more angular techniques and made the project more fun.
